How to Publish This Website Using GitHub Pages

1

Create a GitHub Account

If you don't have one already, go to github.com and sign up for a free account.

2

Create a New Repository

Click on the "+" icon in the top right corner and select "New repository".

Repository name: your-username.github.io (replace with your username)

Make sure the repository is set to Public and initialize with a README.

3

Upload Website Files

Click on "Add file" → "Upload files" and upload this HTML file. You can name it index.html so it loads automatically.

Note: You can also drag and drop the file directly into the repository interface.
4

Enable GitHub Pages

Go to Settings → Pages → Source, and select the main branch.

Click Save, and your site will be published at https://your-username.github.io.

5

Wait for Deployment

It might take a few minutes for your site to be available. Refresh the page after a few minutes.

The URL will be shown in the GitHub Pages section of your repository settings once it's deployed.
6

Share Your Website

Once published, share the URL with your colleagues at American Club Jakarta so they can access your farewell website.
